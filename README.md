Personal Finance Tracker

Live Demo
https://finance-dashboard--dasareddydeeksh.replit.app/

Overview
A full-stack Personal Finance Tracker web application to manage income, expenses, and analyze spending patterns with an interactive dashboard.

## 🚀 Features
- Add and manage income & expenses
- Categorize transactions
- Real-time dashboard analytics
- Visual charts for spending patterns
- Clean and responsive UI

Tech Stack
- React + Vite
- Node.js (Express)
- PostgreSQL
- Recharts
